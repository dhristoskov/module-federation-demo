The repository includes a list of technologies to improve the code quality:

- Formatting (Prettier)
- Testing (Jest)

To improve this process, you can run those quality checks on your local:

Formatting (Prettier) with:

    npm run format:check
    npm run format

Testing (Jest) with:

    npm run test
